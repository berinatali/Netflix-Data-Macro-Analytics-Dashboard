# ExcelPulse: Netflix Data & Macro Analytics Dashboard
![Dashboard Önizlemesi](dashboard-preview.png)

A comprehensive data analytics and business intelligence project built entirely within Excel, featuring advanced data modeling, interactive visualizations, and automated VBA macros for a seamless user experience.

## Project Overview

This project focuses on analyzing Netflix content trends (by genre, release year, and type) to extract meaningful business insights. Designed with a structured multi-sheet architecture, the workbook combines professional dashboard design with automation to deliver a dynamic reporting tool.

## Key Features

* **Interactive Dashboard:** Dynamic charts, slicers, and KPI summaries providing a real-time overview of content metrics.
* **VBA Macro Automation:** Custom macros integrated into the workbook to streamline data navigation, refresh views, and automate user interactions.
* **Dedicated Calculations & Data Modeling:** Clean separation of raw data, calculation layers, and presentation sheets to ensure optimal performance and maintainability.
* **Key Insights:** Actionable takeaways derived from exploratory data analysis (EDA) regarding global streaming trends.
